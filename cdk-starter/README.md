# Welcome to your CDK TypeScript project

This is a blank project for CDK development with TypeScript.

The `cdk.json` file tells the CDK Toolkit how to execute your app.

## Useful commands

- `cdk init app --language typescript` to initate the project
- `cdk bootstrap` to get all dependency
- `cdk synth` compile the code
- `cdk deploy` to deplpy code
- `cdk list` to list all the stack -- this makes local call
- `cdk diff` to check difference between local and deployed code
- `cdk doctor` to check if there is any problem with the cdk
_ `cdk destroy {stackName}` to destroy stack
- `npm run build` compile typescript to js
- `npm run watch` watch for changes and compile
- `npm run test` perform the jest unit tests
- `npx cdk deploy` deploy this stack to your default AWS account/region
- `npx cdk diff` compare deployed stack with current state
- `npx cdk synth` emits the synthesized CloudFormation template
